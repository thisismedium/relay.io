## Getting Started ##

To start developing, you need to install:

  + Node v0.2.6
  + Tokyo Cabinet
  + PostgreSQL

Then, run `./bin/setup.sh` and follow the instructions. This will:

  + Prepare submodule dependencies
  + Add default settings
  + Install DefineJS and renode
  + Do some sanity checks
  + Give you further instructions.

Follow the Setup checklist. When everything is ready, you should be
able to run Relay like this:

    ./bin/develop
